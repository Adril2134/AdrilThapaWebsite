Intercom System

Project Overview

Designed a wired full-duplex intercom system for hotels. The system uses analog circuits and multiplexing techniques to handle multiple connections simultaneously, allowing for noise filtering and clear communication.

The system is designed to be scalable and can be easily expanded to include more rooms or features.

Technical Details

Analog Circuits LM386 Bandpass Filter Multiplexing

Key Features:

  • Noise filtering for clear communication
  • Multiplexing for multiple connections
  • Scalable design for future expansion

Implementation Details

Circuit Design: The system uses a combination of LM386 amplifiers and bandpass filters to handle multiple connections. The multiplexer IC is used to switch between different rooms, while the demultiplexer IC is used to distribute the signal to the corresponding room.

Software Integration: The system uses a microcontroller to control the multiplexer and demultiplexer ICs. The software is designed to handle multiple connections and switch between rooms based on the input signal.

Prosthetic Hand Control System

Project Overview

Developed an advanced prosthetic hand control system using EMG (Electromyography) signals. The system is designed to help people in need of special medical care by providing intuitive control of prosthetic limbs through muscle signals.

The project focuses on extracting and processing EMG signals from forearm muscles to control the movement of a prosthetic hand, making it more accessible and user-friendly for amputees.

Technical Details

EMG ESP32 Signal Processing Band Pass Filter Biomedical Control Systems

Key Features:

  • Real-time EMG signal acquisition and processing
  • Precise movement control through muscle signals
  • Custom band-pass filtering for signal clarity
  • Low-latency response system
  • User-friendly calibration process

EMG Signal Processing

Signal Acquisition:

  • Surface electrodes placed on forearm muscles
  • Raw EMG signal sampling at high frequency
  • Amplification and conditioning of weak signals
  • Noise reduction through hardware filtering

Signal Processing:

  • Band-pass filtering to isolate relevant frequency ranges
  • Signal rectification and envelope detection
  • Feature extraction for movement classification
  • Real-time signal analysis and pattern recognition

Implementation Details

Hardware Components:

  • ESP32 microcontroller for signal processing
  • EMG electrodes for signal acquisition
  • Signal conditioning circuit
  • Motor control system for hand movement

Software Implementation:

  • Custom signal processing algorithms
  • Movement pattern recognition system
